pinoydish API

Github Swagger documentation

Saving a recipe to the database requires an API key, Contact the developer to get one.

For more info regarding on how to create and save a recipe, read the swagger docs

API METHODS

Search recipe by name

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/search?name=Chicken Adobo

Search recipes by first letter

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/search?letter=c

Find recipe by id

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/1

Filter by category

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/filter?category=chicken

Filter by ingredient

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/filter?ingredient=onion

List all categories

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/categories

Get a random recipe

GEThttps://filipino-recipe.herokuapp.com/api/v1/recipe/random
